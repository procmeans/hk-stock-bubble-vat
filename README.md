# 港股可视化 · 估值气泡缸

一个**股市数据可视化**项目。把整个港股市场塞进一个大玻璃缸里,每家公司是一颗气泡——直观地看市场的"贵 / 便宜""大 / 小""涨 / 跌"。

> 🌐 在线访问:**https://procmeans.github.io/hk-stock-bubble-vat/**

这是该可视化系列的第一个维度(市值 × 估值),后续会持续加入更多展示维度(见 [路线图](#路线图))。

---

## 当前维度:市值 × 估值

| 视觉编码 | 含义 |
|---|---|
| 🫧 **气泡大小** | 总市值(越大越显眼,大公司在圈内显示名字) |
| ⬆️ **垂直高度** | PE(TTM)—— 越往上越贵(成长 / 高估值),越往下越便宜(价值) |
| 🪨 **缸底灰色层** | 亏损股(PE ≤ 0)沉淀在底部 |
| 🎨 **颜色** | 单日:按 PE 高低冷→暖;多日:按当日市值涨跌 **红涨绿跌** |

**交互**
- 🖱️ 悬停任意气泡 → 名称、市值、PE、营收、净利、当日涨跌
- 📅 底部**日期滑块**切换历史快照
- ▶️ **播放**:让气泡随时间平滑移动,直观看到市场每天的浮沉
- 🎚️ 一键切换配色(日涨跌 / PE 高低)

## 数据

- **来源**:[AKShare](https://github.com/akfamily/akshare) 聚合的公开行情与财报数据
  - 代码列表:新浪财经
  - 市值 / PE(TTM):百度股市通
  - 营收 / 净利润 / ROE:东方财富(最新年报)
- **覆盖**:全部港股约 2700+ 家(已剔除「－Ｒ」人民币双柜台,避免重复)
- **更新**:每个交易日**港时 20:00** 自动抓取收盘后数据,生成当日快照
- **口径**:市值 / PE 为当日最新;营收 / 净利润为最近一期年报;财务为各公司原始币种

每天一份快照存为 `data/<日期>.json`,`data/manifest.json` 记录所有可用日期。

---

## 自动化

`GitHub Actions`(`.github/workflows/update.yml`)每天定时运行:

```
拉取全部港股代码 → 并发(12 线程)抓取各项指标(~7 分钟)
  → 异常保护(有效记录 < 1500 视为被限流,放弃写入)
  → 写入当日快照 + 更新 manifest
  → 提交并推送(自动触发 GitHub Pages 重新部署)
```

网页为纯静态单页(D3 + Canvas),数据按需从 `data/` 加载,无需后端。

## 本地 / 二次开发

```bash
pip install -r requirements.txt
python fetch_hk.py          # 抓一份当日快照到 data/
# 直接用浏览器打开 index.html(需通过本地服务器以允许 fetch data/)
python -m http.server 8000  # 然后访问 http://localhost:8000
```
